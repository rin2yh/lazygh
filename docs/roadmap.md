# lazygh ロードマップ

## ナビゲーションフロー

```
リポジトリ一覧
  ├→ PR一覧 → PR詳細 → diff表示
  └→ Issue一覧 → Issue詳細
```

---

## Phase 1: MVP（リポジトリ起点のナビゲーション）

読み取り専用。リポジトリ一覧からPR/Issueへ直接遷移できる最小構成。

| 機能 | 詳細 |
|------|------|
| リポジトリ一覧 | 自分が関わるリポジトリ一覧（owner/org別） |
| PR一覧 | リポジトリ選択後にPRリスト表示（状態・タイトル・作者） |
| PR詳細 | PR説明・レビュアー・ラベル・CI状態 |
| PR差分 | ファイル差分のビューア（diffスタイル） |
| Issue一覧 | リポジトリ選択後にIssueリスト表示 |
| Issue詳細 | Issue説明・コメント表示 |

---

## Phase 2: インタラクション

書き込み操作を追加。

| 機能 | 詳細 |
|------|------|
| PRレビューコメント | 行コメントの投稿・返信 |
| PRマージ | squash/merge/rebase選択 |
| Issueコメント | コメント投稿 |
| Issue作成 | タイトル・本文・ラベル指定 |

---

## Phase 3: 拡張機能

nice-to-haveな機能群。

| 機能 | 詳細 |
|------|------|
| GitHub Actions | ワークフロー実行一覧・ログ表示 |
| 通知 | GitHub通知の閲覧・既読管理 |
| リポジトリブラウザ | ファイルツリー閲覧 |
